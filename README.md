### Demo Website ###

An exercise in responsive CSS3 using LESS.

### Dependencies ###

1. Node

### Run ###

```
#!bash

$ node server.js

```

Visit the site at http://127.0.0.1:8081/index.html.

### Features ###

This site was implemented using only CSS3. Features include:

 * image slideshow
 * responsive menu bar
 * menus for each tab
 * responsive images and text
